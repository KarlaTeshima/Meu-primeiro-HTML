Descrição do Curso

Neste curso, você aprenderá e desenvolverá suas habilidades com HTML e CSS , além de ter uma boa base de linguagem de programação Javascript .

Ver conceitos avançados de HTML, incluindo semântica e uma introdução ao SEO (otimização para sites de busca) e acessibilidade .

Já com relação ao CSS, você pode exibir em detalhes todas as propriedades necessárias para criar layouts de páginas e sites na web. Também desenvolvemos um projeto responsivo , ou seja, que funciona bem em todos os dispositivos, celulares, tablets e desktop.

Dê os primeiros passos com Javascript e descubra o poder dessa linguagem, que atualmente é mais usada entre os desenvolvedores da web.

O que você aprenderá:

Poder montar qualquer tipo de site estático, seja ele seu próprio portfólio, seu site pessoal ou uma apresentação de sua empresa na internet Criar sites responsivos de verdade Cria layouts complexos com flexbox e CSS grid Criar efeitos super interessantes com CSS3 Existem requisitos ou pré-requisitos do curso? Nenhum conhecimento prévio. O curso é voltado para iniciantes Para quem é este curso: Todas as pessoas que querem ingressar na área de web design e / ou front-end.
